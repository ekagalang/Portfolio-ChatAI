# portfolio-ai

Portfolio berbasis Next.js App Router dengan AI assistant, integrasi GitHub, dan form kontak via Resend.

## Local development

Install dependency:

```bash
npm install
```

Jalankan development server:

```bash
npm run dev
```

## Environment

Copy `.env.example` ke `.env.local` untuk development atau `.env.production` untuk deployment:

```bash
cp .env.example .env.local
```

## Docker deployment

Panduan deploy dengan Docker Compose tersedia di [DEPLOYMENT.md](./DEPLOYMENT.md).
